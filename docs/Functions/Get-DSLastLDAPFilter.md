# Get-DSLastLDAPFilter

## SYNOPSIS
Returns the last used LDAP filter.

## SYNTAX

```
Get-DSLastLDAPFilter
```

## DESCRIPTION
Returns the last used LDAP filter.
Good for learning or troubleshooting.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-DSLastLDAPFilter
```

Displays the LDAP filter used in the last call to Get-DSObject.
