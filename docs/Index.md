## Installation
PSAD is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/PSAD/).

To Inspect:
```powershell
Save-Module -Name PSAD -Path <path>
```
To install:
```powershell
Install-Module -Name PSAD -Scope CurrentUser
```
